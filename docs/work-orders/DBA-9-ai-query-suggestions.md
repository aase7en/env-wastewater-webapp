# WO-DBA-9: AI query suggestions panel (schema-aware)
Status: open
Lane/files: `frontend/src/components/admin/AiSuggestions.tsx` (new), extends `frontend/src/lib/admin/ai-sql.ts`
Branch: main
Depends on: DBA-8

## Goal + Acceptance
- Side panel in DBAConsolePage that shows **3-5 contextual query suggestions** based on:
  - Schema introspection (read tables + row counts via `pg_class.reltuples` or PostgREST HEAD count)
  - Recent audit_log activity (frequently-touched tables)
  - Recently-run saved queries (tags frequency)
  - Time of year (e.g. "สรุปประจำเดือน" near month-end)
- `lib/admin/ai-sql.ts` extends with `suggestQueries(context): Promise<Suggestion[]>`:
  - Each suggestion = `{ title_th, sql, rationale_th }`
  - Examples: "สรุป kWh รายเดือน 3 เดือนล่าสุด", "การแจ้งซ่อมที่ยังไม่ resolved", "ค่า DO ต่ำกว่า threshold 7 วันล่าสุด"
  - Generated by AI with same PHI filter as DBA-8
- Click suggestion → loads SQL into SqlEditor (DBA-6) for review/run
- Cache suggestions for 5 minutes (avoid re-call on every render)
- Empty state: "ยังไม่มีคำแนะนำ — ลองรัน query หรือ save query ก่อน"

## Verify
- New admin first visit → suggestions show 3+ contextual queries
- Click → loads into editor → run → results
- Suggestion does not reference PHI tables (filtered)
- Cache works (same suggestions within 5min)
